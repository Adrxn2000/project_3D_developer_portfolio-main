# 🚀 3D Developer Portfolio

An interactive 3D developer portfolio showcasing my projects, skills, and experience in a visually immersive way.

## 🌟 Features

- 🧊 3D interactive UI powered by Three.js
- ⚛️ Built with React for scalable components
- 🎬 Smooth animations and transitions
- 📱 Fully responsive design
- 🎯 Project showcase section
- 📬 Contact section

## 🛠️ Tech Stack

- React
- Three.js / React Three Fiber
- Tailwind CSS
- Framer Motion
- Vite

## 📦 Installation

```bash
git clone https://github.com/Adrxn2000/project_3D_developer_portfolio-main.git
cd project_3D_developer_portfolio-main
npm install
npm run dev
